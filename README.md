<h2>Application de gestion de caisse</h2>

<h3>Projet de démonstration de <em>Programmation Orientée Objet</em> en Python avec persistance SQLite.</h3>

<h4>Fonctionnalités :</h4>
  <ul>
      <li>Architecture modulaire (Entités, Composants, Data, Affichage)</li>
      <li>Gestion des tickets avec calcul automatique des totaux</li>
      <li>Sauvegarde des ventes dans SQLite</li>
      <li>Affichage d'un ticket formaté en console</li>
  </ul>
  
<h4>Structure du Projet :</h4>
  <ul>
      <li>main.py : point d'entrée principal de l'application</li>
      <li>/entites : modèles de données (Produit, Vendeur, Categorie)</li>
      <li>/composants : logique métier (Ticket, LigneTicket)</li>
      <li>/data : gestionnaire de base de données</li>
      <li>/affichage : formatage visuel du ticket</li>
  </ul>
